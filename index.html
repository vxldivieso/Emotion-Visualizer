Configuración requerida

Este archivo necesita estar alojado en un servidor https:// para funcionar.
No funciona abierto directamente como archivo local (file://).

Pasos para desplegarlo (gratis):
1. Ve a developer.spotify.com/dashboard → Crea una app.
2. Copia tu Client ID y reemplaza TU_CLIENT_ID_AQUI en la línea del script.
3. Sube este archivo a GitHub Pages, Netlify o Vercel (todos gratis).
4. En tu app de Spotify, agrega la URL del deploy como Redirect URI.
5. Cualquier usuario podrá entrar con su propia cuenta de Spotify.
